## 📌 Project Overview

Bioinformatics is an interdisciplinary field that combines biology, computer 
science, and data science to analyze biological data. Modern biological research 
relies on computational tools, scripting languages, and automated workflows to 
manage large datasets and ensure reproducible research.

This project demonstrates the integration of multiple programming languages, 
version control systems, and automation tools commonly used in bioinformatics 
workflows.

- 🔧 Collaborative coding using **Git and GitHub**

- 📝 Documenting projects using **Markdown**
- ⚙️ Automating workflows using **Bash scripting**
- 📊 Generating structured datasets in **CSV format** for analysis

---

## 📄 Assignment Description

This project was developed as part of the **Continuous Assessment (CA1)** for 
the course **ANA4315 – Introduction to Bioinformatics**.

The goal of the assignment is to simulate a real bioinformatics collaborative 
workflow where multiple contributors develop scripts in different programming 
languages and combine their outputs through automation.

Each group member writes a script in a chosen programming language that prints 
their personal information related to the project.

A **Bash automation script** is then used to:

1. Clone the repository
2. Run all scripts created by group members
3. Capture the outputs from each script
4. Combine the outputs into a structured CSV dataset

This approach demonstrates how different tools and programming environments 
can work together in computational biology research.
